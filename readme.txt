Discord Sound Bot

This Python script utilizes the Discord API to create a functional Discord bot. 
This Discord bot is designed to enhance your server's experience by providing sound 
playback functionality and user management features. With this bot, you can easily
play sound files in your voice channels and manage users efficiently.
This Discord bot extends the functionality of sound playback by introducing queue and playlist features. 
Now, in addition to playing individual sound files, users can create queues and playlists to manage their audio playback more efficiently.

Features:
Sound Playback: Playes audio files using FFMPEG file

Play various sound files in voice channels.
Supports a wide range of audio formats for playback.
Queue Management:

Create a queue of sound files for sequential playback.
Add, remove, or shuffle items in the queue as needed.
Playlist Support:

Create and manage playlists of sound files.
Save, load, and delete playlists easily for repeated use.
User Management:

Manage users with ease, including muting, kicking, or banning members as needed.
Utilize administrative commands to maintain a healthy server environment.
